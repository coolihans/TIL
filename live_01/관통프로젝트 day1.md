## git 기초 흐름

### init -> add -> commit -> push

add : staging area 로 올리는 과정

commit : repository 로 추가하는 과정

- git add

- git status => working tree, staging area 상태 확인하기.
- git commit -m '   '
- git log  => commit 히스토리 보기. version 확인하기
